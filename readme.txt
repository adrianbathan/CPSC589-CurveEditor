////////////////
// HOW TO RUN //
////////////////

1. EXTRACT the project folder 'boilerplate' 
2. COMPILE program using the command line 'make'
3. RUN the program using the command line './boilerplate.out

////////////////
// HOW TO USE //
////////////////

[mouse left click]              :   create a control point
[mouse right click]             :   delete a control point
[mouse left click](hold+drag)   :   moves an existing control point

[arrow up/down]     :   increase/decrease order (k)
[N]                 :   clears and resets screen and values

[U]                 :   shows/hides point on curve at u
[G]                 :   shows/hides geometry of curve at u
[arrow left/right]  :   increases/decreases u

[P]                 :   shows/hides control points
[L]                 :   shows/hides lines between control points

[mouse left click](hold) + [-/=]
                    :   adjusts the weight of a selected point

[V]                 :   shows surface of revolution (when on, other inputs disabled)
[R]                 :   shows/hides axis of rotation
[Comma/Period]      :   rotates axis of rotation
